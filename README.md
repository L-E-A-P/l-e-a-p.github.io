# LEAP Website

Sito web del Laboratorio ElettroAcustico Permanente.

## Setup

Installare le dipendenze Ruby:

```bash
bundle install
```

## Sviluppo Locale

Avviare il server di sviluppo Jekyll:

```bash
bundle exec jekyll serve
```
